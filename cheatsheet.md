# jj Cheat Sheet

### Getting started

| Start a new repo   | Clone an existing repo |
|---------|-------|
| `jj git init` | `jj git clone <url>`    |

### Make a change

| Make a new change   | Describe the content of the current change |
|---------|-------|
| `jj new` | `jj describe -m "description"`    |
| Make a commit   | Chose which parts to commit |
| `jj commit -m "description"` | `jj commit -i`    |
| Make a merge commit | |
| `jj new <id-a> <id-b> <...>` | |
